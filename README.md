# Yet Another Media Player

A desktop  player that plays (fingers crossed) audio from YouTube and SoundCloud with embedded widgets and also local files. It will also record how much time each track was listened to and when (what day...for now...). The listening stats are stored locally.
So it is kind of like Last.fm but without a recommender. Though I hope some day to make it too.

# Installation
The instructions are in [INSTALL.md](INSTALL.md)

# Technical note
This is a work in progress project, that I posted once the database, track selection and addition and basic listening tracking started working. UI still requires work though (as much as everything else).
It is made for Linux, but I am thinking of making it for other platforms as well, including mobile.
